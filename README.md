# Bounded Power Framework (BPF)  
## A system for limiting institutional power

This repository contains a governance system design for constraining institutional authority through structural rules.

It addresses how to:

- limit the growth of institutional power  
- make authority interruptible  
- enforce expiration and renewal  
- ensure actions are auditable  
- preserve reversibility of decisions  

The framework does not define political outcomes.  
It defines how power systems are allowed to operate.

---

## What this is

A structural system for designing institutions that cannot:

- expand without authorization  
- persist without renewal  
- operate without visibility  
- avoid interruption  

This framework can be used to evaluate or design government, institutional, or organizational systems.

## Core idea

Power is treated as a system, not a role.

The framework defines:

- when authority can activate  
- what actions are allowed  
- how actions are recorded  
- how authority is interrupted  
- how authority expires  

The system assumes misuse and designs constraints accordingly.

---

## Key properties

The system is designed to be:

- mechanical  
- interruptible  
- reversible  
- auditable  
- non-expanding
