## The short version

This repository contains a complete worked specification for a public registry of institutional authority — a constitution of structural rules, an operations manual, a registration form, a stop-trigger process, an enforcement system, and roughly twenty example registrations applying the form to authorities ranging from a 940-person homeowners association to a national credit bureau.

None of it is operating. There is no real registry. No institution has filed. The construction exists as a worked specification, presented for discussion.

The experiment asks: *what would it look like if any system exercising authority over people had to publicly declare what it can do, what would stop it, and when its authorization expires?* To make that question answerable rather than rhetorical, the experiment had to be built at the level of detail an actual regime would require. The construction is the argument.

## What this is not

This is not a working registry. The rules are not claimed to be correct. The "Bounded Power Foundation" referenced throughout is a construct of the experiment, not an organization. The "stop trigger" mechanism describes how filings would work if the registry operated, not a process that currently runs. See [HOW-TO-READ-THIS.md](HOW-TO-READ-THIS.md) for the full framing.

## Engagement

Discussion, critique, forking, and use of any part of the construction are all welcome. The repository is not actively maintained as live infrastructure. If something here is useful, take it. If something here is wrong, the wrongness is part of what is available for examination.
