This project defines a structural system for constraining authority so that it remains:

- limited  
- interruptible  
- reversible  
- auditable  
- temporary unless renewed  

It does not define political outcomes.  
It defines how power systems are allowed to operate.

## In one sentence

This is a system that forces institutions to stay limited, expire unless renewed, and remain interruptible from the outside.

---

## In 30 seconds

- Institutions cannot expand without explicit authorization  
- Authority expires unless actively renewed  
- Actions must be recorded and auditable  
- External actors can interrupt harmful actions  
- Systems are designed to be reversible  

---

## What makes this different

This system is not based on oversight or good behavior.

It defines:

- explicit activation conditions for authority  
- required records for every action  
- automatic triggers that can interrupt operations  
- enforced expiration if renewal does not occur  

These mechanisms are designed to operate without relying on trust in actors.
